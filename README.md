# VALYM eSports - webapp

## Requirements

- pnpm
- node 20.9.0

## Setup project

### Install pnpm

```bash
npm i -g pnpm
```

### Install packages

```bash
pnpm i
```

### Force pnpm usage

```bash
npm run preinstall
```

### Configure .env

copy/paste `.env.example` content into `.env` at root of project

## Main commands

### Run development mode

```bash
pnpm dev
```
